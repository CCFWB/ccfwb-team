One-tap deploy 🚀

Enter your GitHub token and tap Deploy. Done in seconds.

First time only — get your token:
  1. Go to github.com/settings/tokens/new
  2. Name it CCFWB Deploy
  3. Expiration → No expiration
  4. Check the repo checkbox
  5. Tap Generate token and copy it
Token is only sent to GitHub's API and never stored.
