One-time setup

Step 1 — Create a free Firebase database

  1. Go to console.firebase.google.com
  2. Open your CCFWB-team project
  3. Tap Databases & Storage in the left menu
  4. Tap Realtime Database
  5. Tap Create Database → pick region → choose Test mode → Enable
  6. Copy the URL shown (e.g. https://ccfwb-team-default-rtdb.firebaseio.com)

Step 2 — Connect

Paste your Firebase Database URL. Share this same URL with teammates.
